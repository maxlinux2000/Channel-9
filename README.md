# Channel-9
Channel-9 transforma tu equipo de radio en un sistema autónomo de gestión. Permite dejar tu estación desatendida, ya que se encarga de vigilar el Canal 9, actuar como Repetidor Loro o funcionar como Secretaría Telefónica. El sistema utiliza inteligencia artificial para transcribir mensajes y responder con voz, liberándote de estar siempre "on air"
